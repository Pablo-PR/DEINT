Cómo acceder a la aplicación

Necesitamos descargar un JDK y un IDE.
Para descargar el JDK vamos a la siguiente dirección: JDK

Seleccionamos la version de nuestro sistema operativo y ejecutamos el archivo .exe que se descarga.
Una vez instalado el JDK, pasamos a seleccionar un IDE, como por ejemplo, eclipse.
Descargar Eclipse IDE

Una vez descargado e instalado, abrimos el IDE y creamos un nuevo proyecto JAVA.
En la carpeta src, pegamos los archivos del código del programa.



Ahora pulsamos en run y se nos ejecutará una ventana llamada "console" en la que ya podemos usar el programa.
